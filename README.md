## About Me
I am a Senior Machine Learning Engineer specializing in Generative AI, embeddings, and RAG architectures. I architect and deploy production-grade ML infrastructure from zero to one, focusing on semantic search, personalization, and context-aware pipelines. 

## Technical Implementation
This portfolio is designed to be lightweight, fast, and highly responsive without relying on heavy frontend frameworks. 
* **Structure:** HTML5
* **Styling:** Custom CSS3 (Flexbox, CSS Grid, CSS Variables, Keyframe Animations)
* **Interactivity:** Vanilla JavaScript (Custom cursor, Intersection Observers for scroll-triggered animations)
* **Hosting:** GitHub Pages

## Running Locally
Because this is a static website, no package managers or build steps are required to view the code locally.
1. Clone the repository: `git clone https://github.com/Swapnil80/Swapnil80.github.io.git`
2. Open the `index.html` file in your preferred web browser.
